To execute the tests please do the following:

1. Copy all Apace CXF libraries into the lib/deploy folder
2. Execute ant test.run
3. There will be test-reports for all tests in the target/testreports directory.
4. TEST-test.endtoend.bookstore.BookstoreEndToEndTest.txt is the main test which executes all Services.
5. Please have a look at this file. You will find log statements which show the order-process.
